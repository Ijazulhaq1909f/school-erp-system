# Al-Samad E.G.H School ERP System

## Setup Instructions

1. XAMPP install karo
2. htdocs mein folder extract karo
3. phpMyAdmin mein `school_erp` database banao
4. `database.sql` import karo
5. `shared/config.php` mein database settings check karo
6. Browser mein `http://localhost/school-erp/` open karo
